Place these files in this /assets folder (exact filenames matter — the site already links to them):

1. profile.jpg
   Your photo, used in the circular hero frame. Square image, at least 500x500px,
   works best. If this file is missing, the site automatically shows a 🏁 emoji
   in its place — nothing breaks either way.

2. favicon.png
   Small square icon (192x192px recommended) shown in the browser tab.

3. social-preview.jpg
   The image shown when your portfolio link is shared on LinkedIn/Twitter/etc.
   Recommended size: 1200x630px.

Project screenshots are optional — the current project cards are text/tag based
(matching your GitHub repos) and don't require images. If you'd like screenshot
previews added to the project cards instead, drop images here named
project-1.jpg, project-2.jpg, etc. and let me know so I can wire them in.
